Python MC

O Python MC é um moto clube formado por um grupo unido de alunos do curso Python Super Dev da ProWay, em Santa Catarina. Mais do que compartilhar o amor pela programação, o grupo encontrou nas estradas uma nova forma de conexão, liberdade e aventura. Unindo a lógica do código com o espírito livre do motociclismo, o Python MC surgiu para fortalecer amizades, trocar experiências e explorar os caminhos catarinenses sempre em boa companhia. Cada linha de código e cada quilômetro rodado representam uma jornada de crescimento e irmandade. Seja no terminal ou na estrada, o Python MC segue acelerando com propósito, paixão e camaradagem.
